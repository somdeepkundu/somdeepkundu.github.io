🎯 Key Improvements

Sensor Fusion
Kalman filtering combines device compass, GPS movement bearing, and magnetometer data for sub-5° heading accuracy.

Multi-Sensor
Automatic fallback: Compass → GPS bearing → Magnetometer. Confidence scoring shows data quality in real-time.

🔬 Purpose

Mobile geo-tagged field capture app correlating phone camera POV with Sentinel-2 satellite imagery for FAO-56 crop growth stage validation using Kc coefficients.

Open Source GitHub Pages Sensor Fusion Kalman Filter Sentinel-2

⚠️ Disclaimer

GPS coordinates, compass heading, altitude and ground footprint estimates depend on device sensor accuracy and may be inaccurate. Data from this app should not be used as the sole basis for agronomic or policy decisions. Sensor performance varies across devices. Always validate with ground truth data.
