Features
========

Sphinx has the following features:

1. **Simple Setup:** Easy to create new projects using `sphinx-quickstart`.
2. **Customization:** Choose from multiple themes and configurations.
3. **Extensions:** Add plugins like autodoc for documenting code.
4. **ReStructuredText:** Write documentation using reST or Markdown.

Check out the Sphinx website for more: https://www.sphinx-doc.org/
