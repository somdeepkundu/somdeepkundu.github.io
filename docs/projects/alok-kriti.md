# Alok Kriti 2.0

![Alok Kriti 2.0](../assets/images/alok-kriti-techconnect.jpg)

## Overview

A geospatial app for night lights analysis using Google Earth Engine. Compare years, generate GIF animations of urban growth, and visualize global luminosity changes over time. Presented at TechConnect, IIT Bombay.

## Details

- **Technologies**: Google Earth Engine, JavaScript, Geospatial, Remote Sensing
- **Purpose**: Night lights analysis and urban growth visualization
- **Features**: 
  - Year-to-year comparison
  - GIF animation generation
  - Global luminosity tracking
