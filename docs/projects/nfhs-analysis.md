# NFHS Spatio-Temporal Data Analysis

![NFHS Analysis](https://opengraph.githubassets.com/1/somdeepkundu/lisa-moran-nss77)

## Overview

Local and Global Moran's I analysis on NFHS (NSS 77th round) data to uncover spatial autocorrelation and clustering patterns in health and socioeconomic indicators across India.

## Details

- **Dataset**: NFHS (National Family Health Survey) - NSS 77th Round
- **Methods**: 
  - Local Moran's I Analysis
  - Global Moran's I Analysis
  - Spatial Autocorrelation Detection
  - Clustering Pattern Analysis
- **Focus Areas**: Health indicators, Socioeconomic indicators across India
- **Technologies**: Python, LISA, Moran's I, Spatial Statistics
